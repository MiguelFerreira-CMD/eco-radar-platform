# 🌐 EcoRadar — Front-end

Esta pasta contém o **front-end** da plataforma EcoRadar, desenvolvido com **Next.js, React e TailwindCSS**.  

O front-end é responsável por:

- Exibir as páginas do projeto
- Consumir a API do back-end via Axios
- Mostrar notícias ambientais
- Permitir envio de denúncias
- Exibir dados de qualidade do ar
- Calculadora de CO₂ e gamificação de pontos

---

## 🧩 Tecnologias

- HTML5  
- CSS3  
- JavaScript / TypeScript  
- React (via Next.js)  
- TailwindCSS  
- Axios (para chamadas à API)  
- React Router
