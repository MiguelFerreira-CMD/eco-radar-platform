# 🖥️ EcoRadar — Back-end

Esta pasta contém o **back-end** da plataforma EcoRadar, desenvolvido com **Node.js e Express**.  

O back-end é responsável por:

- Criar e gerenciar APIs REST
- Armazenar usuários, denúncias e pontuação no MongoDB
- Autenticação de usuários com JWT
- Upload de imagens com Multer
- Servir dados para o front-end

---

## 🧩 Tecnologias

- Node.js  
- Express.js  
- MongoDB / MongoDB Atlas  
- JWT (autenticação)  
- Multer (upload de imagens)  
- Mongoose (ODM)
